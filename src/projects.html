Projects

These are some of my projects during my college journey. Have a breeze through and see my works!

Reviewables

  • Our project is a reviewer application designed to help the user for their study needs. The main function of the application is that users can search for study material or reviewers on their desired subjects.

  • Users can also upload their own reviewers on the app where other users can then view, use, and rate it.

  • Certain innovations or features that make our application different from similar apps have been implemented such as an Al Quiz Maker, and Reviewer Scanner.

Apex Weapon Shop

  • Apex Legends Weapon Shop Program: A GUI-based project designed to test skills in graphical user interfaces, where developers create an interactive environment for users.
  • The project simulates a weapon shop from the game Apex Legends, allowing users to browse through a variety of weapons, view details, and make purchases within the simulated shop.
  • It challenges developers to seamlessly integrate user interactions with a database, ensuring that data such as inventory, user selections, and transactions are accurately managed and updated in real-time.

Data Visualization Using PANDAS

  • Data Visualization using PANDAS: A project aimed at testing and enhancing knowledge in data analysis and visualization techniques, with a focus on using the Pandas library in Python.
  • Users are tasked with working on datasets to clean, process, and analyze data, transforming raw data into meaningful insights. The project involves creating various types of visualizations to present these insights in a clear and understandable format.
  • This project helps developers improve their ability to handle complex data sets and create visual representations, making data more accessible and useful for decision-making processes in real-world scenarios.

Text Based Classroom Management System with CSV

  • Text-based Classroom Management System CSV: A project designed to assess developers' abilities to create and manage a classroom management system using a text-based interface.
  • Users can manage various aspects of classroom data, including student information, attendance tracking, and class scheduling, with all data stored in CSV files for easy access and modification.
  • The project focuses on efficient data handling techniques, requiring developers to implement methods for organizing and retrieving large amounts of classroom data in a user-friendly and reliable manner.

Text Based Library Management System with CSV

  • Text-based Library Management System CSV: This project evaluates proficiency in creating and managing a library system through a text-based interface, focusing on data organization and file handling.
  • Users can add, remove, and search for books, with all records stored in CSV files, which serve as the backbone for data storage and retrieval. The system provides a straightforward yet efficient method for managing library collections.
  • It emphasizes the importance of accurate data handling, allowing developers to practice organizing, accessing, and manipulating information using CSV files, which are common in many real-world applications.
